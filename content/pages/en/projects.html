Physics

[mathjax]My main interests in physics include superconductivity, numerical physics and simulation, quantum mechanics and quantum computing. I did my PhD on experimental quantum computing using superconducting quantum bits. During the thesis, I designed and realized a two-qubit quantum processor with my colleagues and implemented a so-called universal quantum-gate on it. Such a universal gate is comparable to a classical universal logic gate (e.g. NAND) and allows to realize any arbitrary unitary quantum evolution on the processor. To characterize the gate, we performed then quantum process tomography, which consists in measuring the experimental density matrices of the two-qubit register before and after applying the gate operation for various input states. This allows to fully characterize all errors of the gate. Here you see the input and output density matrices for the two input states ( langle 00 | ) and ( langle 01 | ) (quantum state fidelities are shown above the density matrices): Using this quantum gate, we ran the Grover quantum search algorithm on the processor. This algorithm finds the solution to a search problem encoded by a function (f(x)) that operates on states ( xin{00,01,10,11} ) and which is (f(y)=1) for one element (y) and ( f(x)=0 ) for ( x ne y ). Classicaly, the success probability of finding the state (y) after evaluating the function ( f(x) ) only once is bound to 25 %. However, using the quantum Grover search algorithm, this success probability can be 100 %, therefore outperforming any possible classical algorithm.
